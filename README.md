# Poppy LUCIA

Les pièces physiques suivantes permettent de construire un Poppy Lucia.

-  **Support** : le Raspberry Pi et sa caméra est monté sur le Thymio grâce à un support en acrylique 3mm plié.
	- [support](support/) : Des planches SVG et PDF qui permettent de fabriquer 10 supports par découpage laser. Chaque support doit être plié deux fois.
	- [gabarit](gabarit/) : Des planches SVG et PDF pour découper les pièces d'un gabarit qui permet de plier le support à des angles précis.
- **Piste** : les activités UCIA utilisent une piste au sol.
	- [piste](piste/) : une image SVG et PDF au format A1 (594mm × 841mm). Des fichiers supplémentaires contiennent des page A4 pour fabriquer la piste par collage, et pour disposer de panneaux amovibles pour modifier la route.
- **Objets** : les sept fichiers STL dans [objets](objets/) permettent de faire imprimer en 3D les objects reconnues par le NCNN du module `poppy-raspi-thymio`.
